# 1. Exchange_bot
My first ever created bot. 
This is a bot for calculating currency exchange and collecting a client request for my friend.

## Features
- lets users choose the language
- lets users choose the exchange currencies
- asks for the exchange amount
- asks for the city the client from
- shows the final summary
- sends the final order to the host in the private messages

- Allows to dismiss the order in every step

## Tech Stack
- Python
- aiogram

# 3. Find Synagogue bot
This is a bot for finding a Synagogue in every city

There is 2 versions of this bots:
v1 is a version where bot takes data from OSM. It is not always relevant data that is why i've decided to create another version.
v2 is a version where bot takes data from Google Maps but to make this type of connection you have to create an API KEY at Google Cloud.
At the beginning it's free to use but in future i'll have to pay for API KEY usage.

## Features
- lets users choose the city
- bot takes the data from Google maps and shows the exact places where Synagogue is located with the link to Google Maps and Waze

## Tech Stack
- Python
- aiogram

Written by Codex, explained by me
