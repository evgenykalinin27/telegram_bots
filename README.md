# 1. Shai_Exchange_bot
My first ever created bot. 
This is a bot for calculating currency exchange and collecting a client request for my friend.

## Features
- lets users choose the language
- lets users choose the exchange currencies
- asks for the exchange amount
- asks for the city the client from
- shows the final summary
- sends the final order to the host in the private messages

- Allows to dismiss the order in every step

## Tech Stack
- Python
- aiogram
